# Purpose-Agnostic-Agent
A modular, purpose-agnostic AI backend providing context-aware intelligence via RAG and multi-model routing.

Key Features:

Multi-Model Routing: Seamless failover between OpenAI, Anthropic, and local Ollama models using LiteLLM.

Categorized RAG: Knowledge-base management supporting multiple document subjects and namespaces.

MCP Ready: Fully compatible with the Model Context Protocol for easy integration with AI agents and IDEs.

Persona Management: Configurable system prompts to transform the brain into any role (Doctor, Tutor, Assistant).
